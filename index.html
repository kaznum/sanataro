Welcome to Sanataro

This simple web household account book application is built on Ruby on Rails and supports several DBMS products.

Feature

  • Simple kakeibo Web Application.
  • Multi-User capable
  • Balance adjustment
  • Lightweight and beautiful monthly/yearly P/L and B/S charts
  • Credit-card due setting
  • Simple input User-Interface for iPhone.
  • Tweet button! Social Accounting!(Think why this exists and how to enjoy this individually.)
  • See more on a demo page. sanataro.heroku.com/

How to try...

$ cp config/database.yml.sample config/database.yml
$ vi config/database.yml
$ bundle install
$ rake db:setup
$ rails s
